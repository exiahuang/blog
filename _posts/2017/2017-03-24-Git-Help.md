# Git pull 强制覆盖本地文件
```shell
git fetch --all
git reset --hard origin/master
git pull
```

# 拉取代码
```shell
cd D:\Exia.Workspace\myproject\SalesforceXyTools-Chrome
git fetch origin master
git pull origin master
```

# 更新git服务器代码
```shell
git add . -A 
git commit -m "fix"
git push origin master
```
